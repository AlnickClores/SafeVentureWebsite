System Requirements

Minimum:

OS: Android 6.0 and later

Download Size: 12 MB

App Permissions

Camera - Required for capturing images in case of unauthorized access attempts.

Location - Enables tracking user's location for location-based services.

Notification - Allows sending notifications to keep users informed about important events.

Device Admin - Grants ability to detect failed unlock attempts, enhancing security. Foreground Service permission ensures efficient background operation.

User Guide

Upon registration the user will be redirected to the main page of app. Then the app will ask for the following permissions:

- Camera

- Location

- Storage

- Notification

These permissions are required to make sure the app works properly.

Homepage Overview

The homepage displays a map showcasing the user's and their connected friends' locations, with the navigation bar on the upper left side.

Activating Capture Intruder

>Tap the hamburger icon at the top left, then select "Capture Intruder."

>In the "Capture Intruder" section, tap the "Grant Access" button to enable Device Admin Permission.

>Once Device Admin is granted, tap "Choose Email" to select the email address where captured intruder images will be sent.

Adding Friends:

To add friends, access the Socials Tab from the navigation, enter the generated friend code, and upon successful entry, they will be added as your friend.
Friend Code is auto generated upon registration.

Removing Friends:

To unfriend a user, simply click the three dots icon on the same page and select "Unfriend."

Security and Privacy:

Only grant access to the Intruder Catcher feature to trusted individuals. Exercise caution when adding friends and sharing your generated code. Review and adjust privacy settings as needed to protect your personal information.

Disclaimer:

The system aims to improve user experience and implement security measures. However, no system is entirely foolproof, so users should use caution and common sense while utilizing the provided features.

Uninstall the App

Users will not be able to uninstall the app when the device admin is granted, to uninstall the app the users will need to revoke the Device Admin permission from the CaptureIntruder page.

- Go to the CaptureIntruder page, click the “Revoke Permission” button.

- After that users can now uninstall the app.
